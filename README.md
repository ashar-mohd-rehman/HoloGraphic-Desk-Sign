a holographic desk sign which uses usb c as its power source and displays cool holograms. Custom edge-lit acrylic desk sign displaying death star. Powered via a 5V USB-C interface with low-current 470ohm resistors for safe and cool desk lamp. It sits inside a custom designed 3d printed case with slot to change the sign and a push button to switch it on/off
